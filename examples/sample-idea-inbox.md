# Sample Idea Inbox

The purpose of this file is to show how raw ideas can be captured before they are processed into projects, decisions, or next actions.

## Raw Ideas

### Public weather data art

Raw thought:

It would be interesting to turn public weather data into generative art that shows changes in temperature, rainfall, or wind over time.

Possible direction:

Use a public dataset and create visual patterns that are both beautiful and meaningful.

Next question:

What dataset is simple enough for a first experiment?

### Personal reading companion

Raw thought:

A small assistant could help someone capture notes from books, articles, and podcasts, then turn them into themes and project ideas.

Possible direction:

Start with manual notes in Markdown, then later add search and summarisation.

Next question:

What does the simplest useful version look like?
