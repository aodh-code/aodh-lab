# Sample Project Brief

This is fictional example data.

## Project Name

Public Data Art Experiment

## Summary

A small experiment that turns public data into a visual artwork.

## Problem

Many datasets are informative but emotionally flat.

This project explores whether data can be made beautiful while still telling a truthful story.

## User

Someone interested in data, design, and storytelling.

## Why It Matters

Data storytelling can help people notice patterns they might otherwise ignore.

## Core Loop

1. Choose a public dataset
2. Clean the data
3. Identify the story
4. Generate a visual
5. Explain the meaning

## Inputs

- Public dataset
- Visual theme
- Story question

## Outputs

- Static image
- Short explanation
- Reusable script

## Success Criteria

The output should be visually interesting and should communicate something real about the data.

## Risks

- The art may distort the meaning of the data
- The dataset may be too complex for a first version
- The visual may be attractive but not informative

## Next Actions

- Choose a simple dataset
- Create a small Python script
- Generate one visual output
