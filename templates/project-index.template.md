# Project Index Template

This template is for creating a private project index.

Suggested private location:

```text
local/project-index.md
```

Do not commit your real local project index to the public repository.

## Purpose

Use this file to help Aodh Lab decide whether a new idea belongs to:

* An existing project
* A new project
* A one-off task
* An unassigned idea
* Something that should not become a project

## Routing categories

Use these categories when deciding where an idea belongs.

### Existing project

The idea clearly belongs to a project already listed below.

### New project candidate

The idea is substantial enough to become a project, but it does not fit an existing one.

### One-off task

The idea can be completed as a small standalone action.

### Unassigned idea

The idea may become useful later, but it is not ready to route yet.

### Not a project

The idea should not become tracked work.
